# 🎯 Обзор функций веб-дашборда

## 🌟 Основные возможности

### 1. 📊 Интерактивный анализ цен

#### Что делает:
- Сравнивает целевой объект с аналогами
- Рассчитывает справедливую цену
- Определяет степень переоценки/недооценки
- Показывает рыночные тренды

#### Как работает:
```
Целевой объект: 195 млн ₽ (180.4 м²)
├── Цена за м²: 1.081 млн ₽
│
Аналоги (средняя): 0.763 млн/м²
├── С дизайном: 0.792 млн/м²
└── Без ремонта: 0.642 млн/м²

Результат: +36% выше средней по дизайну
Вывод: ПЕРЕОЦЕНЕН
```

### 4. 🧮 Расчет справедливой цены

#### Методология корректировок:

```
БАЗОВАЯ ЦЕНА (средняя по дизайну)
0.792 млн/м²

↓ ПОЛОЖИТЕЛЬНЫЕ КОРРЕКТИРОВКИ

+ Большая площадь (180м² > 150м²)     ×1.05  (+5%)
+ Дизайнерская отделка де-люкс        ×1.08  (+8%)
+ Панорамные виды на залив            ×1.07  (+7%)
+ Премиум локация (Петровский)        ×1.06  (+6%)

↓ НЕГАТИВНЫЕ КОРРЕКТИРОВКИ

- Низкая жилая площадь (22%)          ×0.92  (-8%)
- Только рендеры (неопределенность)   ×0.97  (-3%)

↓ ИТОГОВЫЙ КОЭФФИЦИЕНТ

1.05 × 1.08 × 1.07 × 1.06 × 0.92 × 0.97 = 1.186

↓ СПРАВЕДЛИВАЯ ЦЕНА

0.792 млн/м² × 1.186 × 180.4 м² = 169.5 млн ₽

───────────────────────────────────────────

ТЕКУЩАЯ ЦЕНА:     195 млн ₽
СПРАВЕДЛИВАЯ:     169.5 млн ₽
ПЕРЕОЦЕНКА:       +25.5 млн ₽ (13%)
```
