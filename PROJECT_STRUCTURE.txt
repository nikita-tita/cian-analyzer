ВОЗМОЖНОСТИ
================================================================================

✅ Парсинг:
   • Отдельные объявления
   • Страницы поиска
   • Похожие объявления
   • Все характеристики

✅ Экспорт:
   • JSON (для программ)
   • Markdown (для людей)
   • Готовность к CSV/Excel

✅ Markdown особенности:
   • Красивое форматирование
   • Эмодзи для визуального разделения
   • Интерактивные ссылки
   • Якори для навигации
   • Встроенные изображения
   • Выпадающие блоки
   • Ссылки на карты

✅ Универсальность:
   • GitHub (авто-рендеринг)
   • VS Code (встроенный превью)
   • Notion, Obsidian (импорт)
   • Pandoc (конвертация в PDF/HTML/DOCX)
