# 🚀 Быстрый деплой на Vercel

## ⚠️ Важно: Проблема с оплатой

При попытке деплоя через CLI получили ошибку:
```
Your team has an overdue balance. Please add a valid payment method to reactivate your account.
```

## 🔧 Решение: Деплой через Web UI

### Шаг 1: Оплата (если нужно)

Если хотите использовать существующий аккаунт:
1. Откройте https://vercel.com/teams/nikita-tita-projects/settings/billing
2. Обновите способ оплаты
3. Или используйте Free план (обычно достаточно)

### Шаг 2: Деплой через Dashboard

#### Вариант А: Новый проект

1. Откройте https://vercel.com/new
2. Нажмите **"Import Git Repository"**
3. Найдите и выберите: `nikita-tita/cian-analyzer`
4. Нажмите **"Import"**
5. Vercel автоматически обнаружит:
   - ✅ `index.py` - точка входа
   - ✅ `vercel.json` - конфигурация
   - ✅ `requirements.txt` - зависимости
6. Нажмите **"Deploy"**

#### Вариант Б: Обновление существующего проекта

Если проект уже есть на Vercel:
1. Откройте https://vercel.com/dashboard
2. Найдите проект `cian-analyzer` или `cian`
3. Зайдите в Settings → Git
4. Нажмите **"Redeploy"** или дождитесь автоматического деплоя после push

### Шаг 3: Проверка

После успешного деплоя:
1. Vercel покажет URL типа: `https://cian-analyzer.vercel.app`
2. Откройте URL в браузере
3. Должны увидеть:
   - 🎨 Красивый wizard интерфейс
   - 3 экрана (Парсинг → Аналоги → Анализ)
   - Современный дизайн с floating кнопками

## 📋 Что работает на Vercel

### ✅ Полностью работает:
- **UI/UX** - весь веб-интерфейс
- **API endpoints** - все маршруты
- **Аналитика** - полный движок расчетов
- **Session storage** - in-memory хранилище

### ⚠️ Работает ограниченно:
- **Парсинг** - базовые данные (цена, площадь)
- **Поиск аналогов** - возвращает demo-данные

Причина: Vercel не поддерживает Playwright/браузеры

### 💡 Для полного парсинга:
Используйте Railway (см. [RAILWAY_DEPLOY.md](RAILWAY_DEPLOY.md))

## ⚡ Быстрый старт после деплоя

1. Откройте ваш Vercel URL
2. Введите **любой** URL с Cian.ru (например: `https://www.cian.ru/sale/flat/123/`)
3. Нажмите "Начать анализ"
4. Появятся базовые данные
5. Нажмите "Найти аналоги" → появятся 5 demo-объектов
6. Нажмите "Рассчитать" → увидите полный анализ

**Все работает, просто на demo-данных!** 🎉
