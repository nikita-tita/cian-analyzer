### НЕДЕЛЯ 1: Техническое SEO

#### День 1-2: Базовые исправления

**1. Создать robots.txt:**
```
User-agent: *
Allow: /
Disallow: /api/
Disallow: /metrics
Disallow: /health
Disallow: /static/js/
Sitemap: https://housler.ru/sitemap.xml

User-agent: Yandex
Crawl-delay: 1
```

**2. Добавить Schema.org разметку на главную:**
```json
{
  "@context": "https://schema.org",
  "@type": "RealEstateAgent",
  "name": "HOUSLER",
  "description": "Агентство недвижимости в Санкт-Петербурге. Продажа квартир с полным сопровождением сделки.",
  "url": "https://housler.ru",
  "telephone": "+7-911-029-55-20",
  "email": "hello@housler.ru",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "Санкт-Петербург",
    "addressCountry": "RU"
  },
  "areaServed": "Санкт-Петербург",
  "priceRange": "2% от стоимости"
}
```

**3. Добавить og:image:**
Создать изображение 1200x630px для соцсетей.

#### День 3-4: Оптимизация главной страницы

**Новый Title:**

**Новый H1:**

## РЕАЛИСТИЧНЫЕ ОЖИДАНИЯ

### Что достижимо за 1 месяц:
- Попадание в индекс по всем новым страницам
- Рост по НЧ информационным запросам (позиции 10-30)

- Конкуренция с ЦИАН, Авито, Домклик
- Массовый органический трафик

- Наращивание ссылочной массы
