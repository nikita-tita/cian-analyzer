## 📚 ИНТЕРАКТИВНЫЕ TOOLTIPS

### Как работают:

**1. Термин с ℹ️:**
```
Цена за м² (₽) ℹ️
      ↓ (наведите мышкой)
```

**2. Появляется tooltip:**
```
┌─────────────────────────────────────────────┐
│ Цена за квадратный метр                     │
├─────────────────────────────────────────────┤
│                                             │
│ 💡 Основная метрика для сравнения объектов │
│                                             │
│ 📝 Рассчитывается: Общая цена / Площадь    │
│                                             │
│ 📊 Пример:                                  │
│ Квартира 5,000,000 ₽ / 50 м² = 100,000 ₽/м²│
│                                             │
│ ❓ Почему важно:                            │
│ Позволяет сравнивать квартиры разных        │
│ размеров на одной шкале                     │
│                                             │
└─────────────────────────────────────────────┘
```

**Все термины с tooltips:**
1. **Медиана** - почему не среднее
2. **±3σ** - фильтрация выбросов
3. **Opportunity Cost** - упущенная выгода
4. **ROI** - возврат инвестиций
5. **Price per sqm** - цена за м²
6. **Overpricing** - переоценка
7. **Living Area %** - доля жилой площади
8. **Cumulative Probability** - накопленная вероятность
