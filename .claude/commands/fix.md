# Исправление бага

Баг: $ARGUMENTS

## Алгоритм

### 1. Воспроизведи
- Пойми как воспроизвести баг
- Найди где он проявляется

### 2. Локализуй
```bash
# Поиск по ключевым словам из ошибки
grep -r "keyword" --include="*.py" .
```

### 3. Прочитай код
- Прочитай файл ЦЕЛИКОМ перед изменением
- Пойми как код работает СЕЙЧАС

### 4. Минимальный фикс
- Измени ТОЛЬКО сломанное место
- НЕ рефактори попутно
- НЕ добавляй "улучшения"

### 5. Проверь
```bash
pytest tests/
flake8 .
```

## Правила

- СНАЧАЛА прочитай — ПОТОМ редактируй
- Используй Edit, не Write
- Один баг = одно изменение
- Если не уверен — спроси

## Формат коммита

```
fix: краткое описание что исправлено

Closes #issue (если есть)
```
