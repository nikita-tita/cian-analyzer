# Claude Code Commands

Эта директория содержит custom slash-команды для Claude Code.

## Доступные команды

### `/deploy` - Деплой приложения
Автоматический деплой приложения с выбором режима:
- Development
- Production
- Full Stack (с мониторингом)

### `/status` - Статус деплоя
Проверка текущего состояния:
- Запущенные контейнеры
- Health check
- Последний деплой
- Использование ресурсов

### `/logs` - Просмотр логов
Показывает логи приложения

### `/stop` - Остановка сервисов
Останавливает все запущенные контейнеры

## Использование

В Claude Code просто напишите:
```
/deploy
```

Claude выполнит команду и покажет результат.
